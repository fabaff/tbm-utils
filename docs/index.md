# tbm-utils

tbm-utils is a commonly-used set of utilities used by me (thebigmunch).


## Getting Started
===============

Install tbm-utils with `pip <https://pip.pypa.io/en/stable/>`_.

``` shell
$ pip install -U tbm-utils
```
